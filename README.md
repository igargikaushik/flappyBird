# Flappy Bird Platformer Game

## Description
This repository contains the source code and assets for a platformer genre Flappy Bird game implemented in C# using UNITY.

## Features
- Classic Flappy Bird gameplay with a platformer twist.
- Responsive controls for a smooth gaming experience.
- Score tracking and high score display.
